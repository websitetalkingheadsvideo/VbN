If you don't see animations:

  1. Make sure you're using a modern browser (Chrome, Firefox, Edge)
  2. Try hard refresh: Ctrl+Shift+R (Windows) or Cmd+Shift+R (Mac)
  3. Check browser console (F12) for errors
